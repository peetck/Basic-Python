ทําไมต้องเป็น Python?


หลากหลาย

  • มี Library พร้อมให้ใช้เยอะมาก ไม่ว่าจะเป็น Library ด้านการคำนวน เช่น numpy,scipy หรือด้านที่ใช้งานเกี่ยวกับ Graphic เช่น pyOpenGL, Python Image Library, pyOpencv เป็นต้นและเนื่องจากเป็นภาษาที่เปิดกว้างจึงมี library เพิ่มมากขึ้นเรื่อยๆ

เรียนรู้ง่าย

  • เนื่องจาก Python เป็น High-Level Programming Language และเป็นมิตรกับคนเขียนเพราะไวยากรณ์ไม่ซับซ้อน และเข้าใจง่าย จนมีคนกล่าวไว้ว่า "ถ้าคุณเข้าใจภาษาอังกฤษ คุณก็เข้าใจภาษา Python"
  • สําหรับคนที่ไม่เคยเขียนโปรแกรม สามารถใช้พื้นฐานการเขียนโปรแกรมภาษา Python ไปต่อยอดในการเขียนภาษาอื่นได้

มาแรง

  • ปัจจุบัน Python เป็นที่นิยมมาก ทําให้มีกลุ่มคนผู้ใช้งานจํานวนมากที่พร้อมให้ความช่วยเหลือและตอบคําถามเมื่อมีปัญหา และสําหรับประเทศไทยมีสถานอุดมศึกษาหลายแห่งได้บรรจุภาษา Python เป็นวิชาหลักสูตรด้านการเขียนโปรแกรมคอมพิวเตอร์แล้ว
  • เป็นที่ต้องการของตลาด เงินเดือนเฉลี่ยของ Python Developer ที่ประเทศอเมริกาสูงถึง 117,298 เหรียญ (2018)
