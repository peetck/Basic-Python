Operators (ตัวดำเนินการ)

Operators หรือตัวดำเนินการ คือสิ่งที่เอาไว้ใช้ในการกระทําการทางคณิตศาสตร์ต่างๆกับข้อมูลครับ ซึ่งในภาษา Python จะมีหลักๆอยู่ด้วยกันตามนี้เลยครับ

การดําเนินการ สัญลักษณ์ หน้าที่
Addition + การบวก
Subtraction - การลบ
Multiplication * การคูณ
Division / การหาร
Floor Division // หารแบบตัดเศษ
Modulo % หารเพื่อเอาเศษ
Exponentiation ** การยกกําลัง

ซึ่งตัวดําเนินการพวกนี้มีไว้เพื่อให้เราสามารถ คํานวณทางคณิตศาสตร์ในโปรแกรมของเราได้ครับยกตัวอย่างเช่น

ซึ่งถ้าเราสังเกตุนะครับ หากเราทําการดําเนินการ Float กับ Int หรือ Float กับ Float เนี่ยสิ่งที่จะได้ออกมาจะกลายเป็น Float ครับ

สําหรับการดําเนินการกับ String เราจะขอเก็บไว้พูดในตอนที่เป็นหัวข้อของ String นะครับ

ซึ่งในภาษา Python จะใช้มาตรฐานสิ่งที่เรียกกันว่า PEMDAS ครับซึ่งมันก็คือ ลําดับการกําหนดว่าจะทําอะไรก่อนอะไรหลังครับซึ่งลําดับความสําคัญจะเป็นไปตามนี้เลย

ลําดับความสําคัญ การดําเนินการ ทิศทาง
1 วงเล็บ ซ้ายไปขวา
2 ยกกําลัง (**) ขวาไปซ้าย
3 คูณ, หาร, หารแบบตัดเศษ, หารเพื่อเอาเศษ (*, /, //, %) ซ้ายไปขวา
4 การบวก, การลบ (+, -) ซ้ายไปขวา

ยกตัวอย่างเช่น

ลําดับของการดําเนินการในรูปแรกจะเป็นตามนี้ครับ (ตัวอักษรสีแดงจะเป็นการบอกว่าคํานวณก่อนครับ)

  • 20 ** 2 + (2 * (80 / 5) ** 5) % 5 - 2 + 8
  • 20 ** 2 + (2 * 16 ** 5) % 5 - 2 + 8
  • 20 ** 2 + (2 * 1048576) % 5 - 2 + 8
  • 20 ** 2 + 2097152 % 5 - 2 + 8
  • 400 + 2097152 % 5 - 2 + 8
  • 400 + 2 - 2 + 8
  • 402 - 2 + 8
  • 400 + 8
  • 408

ลําดับของการดําเนินการในรูปที่สองจะเป็นตามนี้ครับ (ตัวอักษรสีแดงจะเป็นการบอกว่าคํานวณก่อนครับ)

  • 20 + (5 + 2 * 5 ) / 5
  • 20 + (5 + 10) / 5
  • 20 + 15 / 5
  • 20 + 3
  • 23
