String

String คือลำดับหรือกลุ่มของตัวอักษรที่ ซึ่งใน Python จะอยู่ภายใต้ (" ") หรือ (' ') เท่านั้นครับ

String สามารถ + หรือ * ได้ครับ โดยจะ + ได้แค่กับ String ด้วยกันเท่านั้นครับและจะ * ได้กับแค่ตัวเลขเท่านั้นครับ

Index String

เราสามารถเลือกว่าจะเข้าถึง ตัวอักษรใน String ตัวไหนลำดับที่เท่าไรได้ครับผ่านทาง Index โดยใช้ []

Slicing Strings

เราสามารถกำหนดขอบเขตการเข้าถึง String ได้ครับโดยใช้การ Slice โดยกำหนดขอบเขตให้มันครับโดยใช้ [start : stop : step]

โดยหากเราไม่ใส่ start ค่าจะเป็น 0 หากเราไม่ใส่ stop จะเป็น ขนาดของ String และ step จะเป็น 1 ครับ

String method

String จะมี method สำหรับจัดการข้อมูลครับและอำนวยความสะดวกในการเขียนโปรแกรมเป็นอย่างมาก และยังช่วยลดเวลาในการเขียนโปรแกรม เราสามารถเรียกใช้โดย พิมพ์ (.) หลัง String แล้วตามด้วยชื่อ method ครับ

String method มีอะไรบ้างตัวอย่าง

String method หน้าที่
.upper() ทำการเปลี่ยนตัวพิมพ์เล็กเป็นตัวพิมพ์ใหญ่
.lower() ทำการเปลี่ยนตัวพิมพ์ใหญ่เป็นตัวพิมพ์เล็ก
.swapcase() เปลี่ยนพิมพ์เล็กเป็นใหญ่ และพิมพ์ใหญ่เป็นเล็ก
.capitalize() เปลี่ยนตัวแรกให้เป็นตัวพิมพ์ใหญ่
.find() หาตำแหน่งของคำใน string (จะเอาอันที่เจอก่อน)
.replace() แทนที่คำด้วยคำที่เรากำหนดเอง
.count() นับว่าใน string มีคำนี้กี่คำ
.isalnum() เช็คว่ามีตัวอักษรหรือตัวเลข หรือเปล่า
.isalpha() เช็คว่าทุกตัวเป็น alphabet หรือป่าว
.isupper() เช็คว่าทุกตัวเป็นพิมพ์ใหญ่รึป่าว
.islower() เช็คว่าทุกตัวเป็นพิมพ์เล็กรึป่าว
.isdigit() เช็คว่าทุกตัวเป็นตัวเลขรึป่าว
.center() padding ด้วยคำและขนาดที่ต้องการ
.strip() ลบคำซ้ายขวา
.lstrip() ลบคำแต่ด้านซ้าย
.rstrip() ลบคำแต่ด้านขวา
เรามาดูตัวอย่างการใช้งานกันครับ
