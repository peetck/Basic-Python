String

String คืออะไร?

String คือลำดับหรือกลุ่มของตัวอักษรที่ ซึ่งใน Python จะอยู่ภายใต้ (" ") หรือ (' ') เท่านั้น

---------ใส่รูป-----------

String สามารถ + หรือ * ได้ ถึงแม้ String จะไม่ใช่ตัวเลขแต่ก็สามารถ + หรือว่า * กันได้เช่น

---------ใส่รูป-----------

Index String

เราสามารถเลือกว่าจะเข้าถึง ตัวอักษรใน String ตัวไหนลำดับที่เท่าไรได้ผ่าน Index โดยใช้ []

---------ใส่รูป-----------

Slicing Strings

เราสามารถกำหนดขอบเขตการเข้าถึง String ได้โดยใช้ การ Slice โดยกำหนดขอบเขตให้มันเช่น [start : stop : step]

---------ใส่รูป-----------

String method

String จะมี method สำหรับจัดการข้อมูลและอำนวยความสะดวกในการเขียนโปรแกรมเป็นอย่างมาก และยังช่วยลดเวลาในการเขียนโปรแกรม เราสามารถเรียกใช้โดย พิมพ์ (.) หลัง String แล้วตามด้วยชื่อ method เช่น

String method มีอะไรบ้างตัวอย่าง เช่น

  • .upper() ทำการเปลี่ยนตัวพิมพ์เล็กเป็นตัวพิมพ์ใหญ่
  • .lower() ทำการเปลี่ยนตัวพิมพ์ใหญ่เป็นตัวพิมพ์เล็ก
  • .swapcase() เปลี่ยนพิมพ์เล็กเป็นใหญ่ พิมพ์ใหญ่เป็นเล็ก
  • .capitalize() เปลี่ยนตัวแรกให้เป็นตัวพิมพ์ใหญ่
  • .title() เปลี่ยนให้เป็นเหมือน title
  • .find() หาตำแหน่งของคำใน string (จะเอาอันที่เจอก่อน)
  • .replace() แทนที่คำด้วยคำที่เรากำหนดเอง
  • .count() นับว่าใน string มีคำนี้กี่คำ
  • .isalnum() เช็คว่ามีตัวอักษรหรือตัวเลข หรือเปล่า
  • .isalpha() เช็คว่าทุกตัวเป็น alphabet หรือป่าว
  • .isupper() เช็คว่าทุกตัวเป็นพิมพ์ใหญ่รึป่าว
  • .islower() เช็คว่าทุกตัวเป็นพิมพ์เล็กรึป่าว
  • .isdigit() เช็คว่าทุกตัวเป็นตัวเลขรึป่าว
  • .center() padding ด้วยคำและขนาดที่ต้องการ
  • .strip() ลบคำซ้ายขวา
  • .lstrip() ลบคำแต่ด้านซ้าย
  • .rstrip() ลบคำแต่ด้านขวา

ยังมี method อีกมากมายที่เรายังไม่ได้พูดถึง สามรถดูเพิ่มเติมได้ที่ Python String Methods
