String Format

ในบางครั้งเราสามารถแสดงผล String ได้ด้วยการ Format ครับ โดยใช้เครื่องหมาย % เพื่อที่จะให้ออกมาตามที่เราต้องการครับ โดยจะมีรูปแบบ % ดังนี้เลย

รูปแบบ สําหรับข้อมูลประเภท
%s String
%d Integer
%f Float
%e เลขแบบวิทยาศาสตร์

เรามาดูตัวอย่างการใช้งานกันเลยครับ

แล้วถ้าเราอยากทํามากกว่า 1 ตัวก็สามารถทําได้ครับ โดยการ % จะทําการดึงตามลําดับจากซ้ายไปขวาครับ ยกตัวอย่างเช่น

จากรูป %s อันแรกจะแทนตัวแปรที่มีชื่อว่า name ครับและ %d จะแทนตัวแปรที่มีชื่อ age ตามลําดับครับ

ในบางครั้งเวลาเราอยากแสดงผลทศนิยม(Float) โดยใช้การ Format สิ่งที่เราจะได้จะเป็นทศนิยม 6 ตําแหน่งเสมอครับ ยกตัวอย่างเช่น

โดยหากเราสามารถกําหนดได้ครับว่าจะเอากี่ตําแหน่ง โดยใช้ "%.(ตําแหน่ง)f" ครับ ยกตัวอย่างเช่น

และเราสามารถทําสิ่งที่เรียกว่า String Alignment ได้ครับ เช่น เราอยากให้เว้นทางด้านซ้าย 10 ช่อง และค่อยตามด้วยข้อความ โดย

หากเราใส่เลขจํานวนเป็นบวก ก็จะเว้นซ้ายครับ

หากเราใส่เลขจํานวนเป็นลบ ก็จะเว้นขวาครับ

ยกตัวอย่างเช่น

แต่ถ้าหากเราใช้กับ Integer เราสามารถใส่ 0 ก่อนจะกําหนดขนาดได้ครับจะเป็นการเติม 0 แทนช่องว่าง ยกตัวอย่างเช่น

เรามาดูตัวอย่างการใช้งานแบบรวมๆกันอีกรอบครับ
