List

List คืออะไร?

List เนี่ยให้เรามองว่าเป็นเหมือนกล่องๆหนึ่งครับที่เราสามารถใส่ชนิดข้อมูลหลากหลายชนิดลงไปเก็บไว้ได้ โดยจะคั่นแต่ละ items ใน list ด้วยเครื่องหมาย , และจะถูกครอบด้วย [ และ ] ค่าของแต่ละตัวใน list สามารถเปลี่ยนแปลงไดhไม่เหมือนกับ String

ยกตัวอย่างเช่น

อย่างในรูปนี่หมายความเป็น ตัวแปร my_list เป็น list ที่มีสมาชิก 5 ตัวครับ

วิธีสร้าง List

สามารถทําได้หลักๆอยู่ 3 แบบครับ

  • ใช้เครื่องหมาย []
  • list()
  • [..สมาชิกแต่ละตัว..]

ยกตัวอย่างเช่น

เราสามารถ + หรือ * List ได้ครับโดยที่

  • List สามารถ + กันได้จะต้องเป็น list กับ list เท่านั้น
  • List สามารถ * กันได้จะต้องเป็น list กับ int เท่านั้น

ยกตัวอย่างเช่น

การเข้าถึง Items ใน List

การเข้าถึง items แต่ละตัวใน list วิธีจะคล้ายๆกับ String ครับคือ การ Index ตำแหน่งของ items ที่เราต้องการและเรายังสามารถ Slicing ได้อีกด้วยครับ ยกตัวอย่างเช่น

List is mutable

List สามารถเปลี่ยนแปลงค่าได้ครับไม่เหมือนกับ String โดยเราสามารถ index แล้ว = ค่าที่เราต้องการได้เลยครับ เช่น

เรายังสามารถใช้ Loop กับ List ได้ครับ ยกตัวอย่างเช่น

เรายังสามารถใช้ In เช็คว่ามีสมาชิกตัวนั้นอยู่ใน List หรือเปล่าก็ได้ครับ ยกตัวอย่างเช่น

การเพิ่ม สมาชิกเข้าไปใน List

  • วิธีที่ 1 (.append) : คือการนําสมาชิกที่เพิ่มเข้าไปต่อท้ายใน list ครับ
  • วิธีที่ 2 (.extend) : จะมีข้อแตกต่างจาก .append()> ตรงที่เราต้องใส่อะไรที่เป็น iterable(เช่น list, string) และมันจะกระจายสมาชิกแต่ละตัวเข้าไปต่อท้ายใน list ให้ครับ
  • วิธีที่ 3 (.insert) : จะเป็นการเพิ่มโดยที่เราสามารถกําหนดตําแหน่งได้ว่าจะให้เพิ่มตรงตำแหน่งไหนครับ (ถ้า append & extend จะเป็นการนำไปต่อท้าย)

วิธีลบสมาชิกออกจาก List

  • วิธีที่ 1 (.remove) : หากอยากอยากลบสมาชิกที่มีค่าเป็นอะไรก็กําหนดได้เลยครับ
  • วิธีที่ 2 (.pop) : จะแตกต่างจาก remove ตรงที่เราสามารถลบได้จากตำแหน่งของสมาชิกนั้นครับ และจะทําการคืนค่าออกมาด้วย

วิธีการ Copy List

  • วิธีที่ 1 (.copy) : การที่เราจะ copy list นั้นหากเราบอกให้ lista = listb หากเราบอกแบบนี้ถ้าเราเปลี่ยนแปลงค่าใน b ค่าใน a จะเปลี่ยนตามไปด้วยครับ เราจึงต้องใช้ method ของ list ที่ชื่อว่า .copy()
  • วิธีที่ 2 (list) : อีกวิธีนึงคือใช้ built-in function ที่ชื่อว่า list() ครับเหมือนเวลาเราประกาศ list ได้เลย

ความแตกต่างระหว่าง list(str) และ [str]

  • หากเรา list(str) มันจะเหมือนกับการที่เรามี list เปล่า แล้วเรา extend string เข้าไปครับ
  • หากเรา [str] มันจะเหมือนกับการที่เรา append นั่นเองครับ

เราสามารถใช้ len() กับ list ได้เช่นกันครับ

เราสามารถใช้ max(), min() กับ List ได้เช่นกันครับ

เราสามารถใช้ list() กับ range() ได้

String มี method ตัวนึงที่ชื่อว่า .split() ครับมันสามารถเปลี่ยนจาก String เป็น list ได้โดยแบ่งได้ด้วยคำที่ต้องการหากไม่ใส่ค่า default ของมันคือ ช่องว่างครับ

List Methods

List Methods หน้าที่
.append()ทำการเพิ่ม(ต่อท้าย)เข้าไปใน list
.extend()ทำการเพิ่มเหมือนกัน (อธิบายความแตกต่างไปแล้วนะครับจากข้างบนเลย)
.clear()ทำการลบสมาชิกทุกตัวใน list (ทำให้เป็น list เปล่าๆ)
.copy()Copy List
.count()คล้ายๆ string เป็นการนับว่ามีสมาชิกตัวนี้ใน list อยู่กี่ตัว
.index()คล้ายๆ .find() ของ String ครับจะทำการเริ่มหาจากซ้ายไปขวาและบอกตำแหน่งแรกที่เจอมา
.insert()ทำการเพิ่มสมาชิกโดยกำหนดตำแหน่งที่ต้องการเพิ่มได้ครับ
.pop()ลบไอเท็มใน list ออกโดยกำหนดจาก index (จะ return ค่าที่ลบออกมาด้วย)
.remove()ลบไอเท็มใน list ออกโดยกำหนดจากค่าสมาชิก
.reverse()เป็นการย้อน list ครับเช่นจาก [1, 2, 3] เป็น [3, 2, 1]
.sort()เป็นการเรียง list จะอธิบายในหัวข้อต่อไปครับ

การเรียงลําดับสมาชิกใน List (Sort list)

แบบที่ 1 .sort() จะมี ค่าให้เราใส่ สองค่า คือ reverse และ key ครับโดยที่ค่า default ของ reverse คือ False (น้อยไปมาก) ส่วน key คือเราสามารถกำหนดได้ว่าจะเรียงจากอะไรครับ

หากเราอยากได้ค่าจาก มากไปน้อย

ส่วน key เราสามารถใส่เป็น function ที่เราเขียนเองก็ได้ครับหรือ built in function ก็ได้ เช่น

แบบที่ 2 Sorted() Sorted เป็น built-in function ครับหลักการทำงานเหมือนกับ sort แต่ต่างกันตรงที่ sort เป็น method และจะเปลี่ยนแปลงตัวเองครับแต่ sorted จะทําการเรียง และ return ออกมาเป็นอีก list นึงครับ
