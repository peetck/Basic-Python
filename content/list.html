List

List คืออะไร?

List ให้เรามองว่าเป็นเหมือนกล่องๆหนึ่งที่เราสามารถใส่ชนิดตัวแปรหลากหลายชนิดลงไปเก็บไว้ได้ โดยจะคั่นแต่ละ items ใน list ด้วยเครื่องหมาย ,

และจะถูกครอบด้วย [ และ ] ค่าของแต่ละตัวใน list สามารถเปลี่ยนแปลงไดhไม่เหมือนกับ String ถ้าหากเคยเขียนภาษาอื่นมาบ้างมันก็คือ Array ของภาษาอื่นนั่นเอง

List Methods

คราวนี้เรามาดู list method ดีกว่าว่ามีอะไรบ้างวิธีเรียกใช้ก็อย่างที่รู้กันครับ list[].methods()

  • .append() ทำการเพิ่ม(ต่อท้าย)เข้าไปใน list
  • .extend() ทำการเพิ่มเหมือนกัน (อธิบายไปในหน้าก่อนๆ ข้างต้น)
  • .clear() ทำการลบทุกตัวใน list (ทำให้เป็น list เปล่าๆ)
  • .copy() อย่างที่บอกไปคือเอาไว้ copy list
  • .count() คล้ายๆ string เป็นการนับว่ามี Items ใน list กี่ตัว
  • .index() คล้ายๆ .find() ของ String จะทำการเริ่มหาจากซ้ายไปขวาและบอกตำแหน่งแรกที่เจอมา
  • .insert() ทำการเพิ่ม Items โดยกำหนดตำแหน่งที่ต้องการเพิ่ม Items ได้
  • .pop() ลบไอเท็มใน list ออกโดยกำหนดจาก index (จะ return ค่าที่ลบออกมาด้วย)
  • .remove() ลบไอเท็มใน list ออกโดยกำหนดจากค่า
  • .reverse() เป็นการย้อน list เช่นจาก [1, 2, 3] เป็น [3, 2, 1]
  • .sort() เป็นการเรียง list จะอธิบายในลำดับต่อๆไป

Sort list

แบบที่ 1 .sort() จะมี ค่าให้เราใส่ สองค่า คือ reverse และ key โดยที่ค่า default ของ reverse คือ False (น้อยไปมาก) ส่วน key คือเราสามารกำหนดได้ว่าจะเรียงจากอะไร
